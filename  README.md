# Kara Kala - Modern E-commerce Theme

A high-performance, responsive, and feature-rich HTML/CSS/JS template inspired by leading e-commerce platforms like **Digikala**. This project is specifically designed to be converted into a fully functional WordPress & WooCommerce theme.

## 🚀 Key Features

- **Digikala-Inspired UI**: Familiar and user-friendly interface optimized for Iranian users.
- **Responsive Design**: Fully compatible with Desktop, Tablet, and Mobile devices.
- **Custom CSS Variables**: Built using a modern design system with reusable tokens and CSS variables.
- **Dynamic Components**:
    - Multi-level Mega Menu
    - Hero Sliders & Product Carousels (Powered by Swiper.js)
    - Advanced Product Filters
    - AJAX-ready Auth Modals & Mini-Cart
    - Specialized Shop & Product Single pages
- **Typography**: Optimized with Persian standard web fonts (IRANYekan).
- **No Heavy Frameworks**: Lightweight and fast, using vanilla JS where possible for better performance.

## 📂 Project Structure

- `/css`: Modular stylesheets (Auth, Cart, Checkout, Single Product, etc.)
- `/js`: Specialized scripts for UI interactions (Mega-menu, Search, Sliders).
- `/fonts`: Standard Persian web fonts.
- `/img`: Optimized SVG icons and placeholders.
- `index.html`: Homepage with "Amazing Deals" and various category sections.

## 🛠 Tech Stack

- **HTML5** & **CSS3** (Custom Properties / Variables)
- **JavaScript** (ES6+)
- **Swiper.js** (For touch-enabled sliders)
- **FontAwesome** (For iconography)
